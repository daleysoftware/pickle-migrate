# pickleMIGRATE

*Work in progress.*

`pickleMIGRATE` is a lightweight and simple migration tool for the `pickleDB`
key-value store.

## Usage

Using `pickleMIGRATE` is easy!

```
>>> db = picklemigrate.load_and_migrate('example.db', False)

```
